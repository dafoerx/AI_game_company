## 二、题材选择原因

经过本次发散和调研，最终没有选择传统农场/餐厅/商店经营，而是选择“AI 游戏公司经营模拟”，核心原因如下：

1. 更适合用 Web 低成本表达
2. 更贴近当前项目本身的 AI 协作命题
3. 办公室、人员、项目、营销、服务器等模块天然适合拆成可验证系统
4. 原型阶段更容易用数值和决策反馈验证乐趣，而不强依赖重美术和演出
