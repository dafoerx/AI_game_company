# 沸腾掌柜：火锅店经营MVP

## 项目简介

一款面向Web端的火锅店运营模拟原型，聚焦“备货—接待—出餐—结算—复盘”的单店经营体验，让玩家在有限时间内平衡食材成本、翻台效率与顾客满意度，感受火锅店从保本到盈利的经营决策乐趣。

## 视觉风格

2D俯视角卡通风格，暖色系（红/橙）突出火锅氛围；界面以清晰信息层级为主，功能按钮大而直观，数值反馈（收入、满意度、库存）强调可读性，优先低成本高辨识度资源方案。

## 核心玩法

玩家在营业前进行采购与备料（锅底、肉类、蔬菜、饮品），营业中处理顾客入座、点单、上菜与翻台，并根据高峰时段调配有限资源；营业结束后进行日结复盘，查看利润、浪费和口碑变化，再将收益投入次日采购与小幅能力提升，进入下一轮经营。
