# 《中转之家：放心去爱》

## 项目简介

在 Web 端经营一家流浪动物中转之家，玩家通过观察行为、陪伴互动与情绪判断，逐步修复动物的创伤与信任问题，再借助社交发布为它们匹配合适家庭；在产生深厚感情后学会“适时放手”，并通过回访见证它们的新生活，传递治愈与被治愈的温柔内核。

## 视觉风格

温暖绘本感 2D 手绘风格，低饱和治愈配色；动物表情演出细腻，界面偏轻量卡片化，过场采用照片墙与短视频拼贴，突出“记录与告别”的情绪体验。

## 核心玩法

玩家每日接收流浪动物并查看其背景档案，先通过观察行为与互动反馈识别“心结”，再安排陪伴与小游戏提升信任、降低应激；当动物达到可领养状态后，制作并发布征婚视频卡片，匹配潜在家庭并做送养决策；送养后获得回访照片/视频，解锁新事件与新救助个体，进入下一轮“接收-疗愈-放手”的循环。
