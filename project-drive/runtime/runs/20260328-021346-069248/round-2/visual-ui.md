   - 运行状态：`day / minuteOfDay / gameState`  
   - 核心数据：`cash / satisfaction / inventory摘要`  
   - 存档反馈：`schemaVersion / lastSaveAt / loadResult(含错误提示)`  
4. **暂停/继续必须提供双重可见反馈**：按钮文案状态变化 + 全局状态Badge（Running/Paused），避免“操作已触发但用户不确定是否生效”。  
5. **异常反馈必须用户可读**：损坏JSON、版本不匹配、存储失败不能只在控制台输出，需在面板上给出明确结果文案，支撑QA截图留证。
